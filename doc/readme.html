Thank you for downloading eAthena 1.0.0 RC5
eAthena is an emulator for Ragnarok Online servers. It's written in C, and is cross-platform. The officially supported platforms are Win32 and Linux.

We hope you enjoy using it!

eAthena is licensed under the GPL, meaning it's open source :) There is no public CVS provided, however.
